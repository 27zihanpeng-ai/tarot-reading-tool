# 🔮 Cyberpunk Tarot Reading Tool

AI-powered tarot reading with a cyberpunk aesthetic and bilingual support (English/Chinese).

## ✨ Features

- **Cyberpunk Design**: Deep black + neon blue + gold aesthetic
- **78 Tarot Cards**: Complete deck with meanings
- **Multiple Spreads**: Single card, 3-card, Celtic Cross
- **AI Interpretations**: Powered by Groq API
- **Bilingual**: English & Chinese support
- **Content Moderation**: Automatic filtering
- **No Login**: Instant access

npm run dev
